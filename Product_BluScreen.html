BluScreen Smart Display

BluScreen is an innovative IoT-based LED matrix project combining technology, creativity, and connectivity into one smart display system.

Overview

BluScreen Project is a smart LED matrix display system that connects to a website through Wi-Fi and Bluetooth Low Energy (BLE) to perform multiple interactive functions. It can show real-time time, date, weather, and temperature, as well as display custom text entered by the user. BluScreen can also start timers and remind users of tasks on command. All these features are controlled remotely from a web interface, making it a versatile IoT-based display board that blends convenience, connectivity, and creativity.
